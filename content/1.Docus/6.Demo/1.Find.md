# Find a country by label

Method: <code>POST</code>

Endpoint to get countries by label with this method

Endpoint:
```
/api/countries/find
```
Body:
```
{
    "country": "France" // string
}
```

Response:
```
{
    "countries": [
        {
            "_id": "FR",
            "label": "France"
        }
    ]
}
```
